*CineChain*

## Description

CineChain is an automated analysis system that estimates theatre occupancy and movie revenue by analyzing seat booking data from online ticketing platforms.

It navigates to currently running movies in theatres, accesses the seat booking interface, captures seating layouts, and computes approximate audience count and revenue per show.

## Features

### Seat Occupancy Detection

* Automatically opens movie show listings
* Navigates to seat selection pages
* Captures screenshots of seating layouts
* Counts occupied seats to estimate audience size

### Revenue Estimation

* Uses detected seat occupancy
* Applies ticket price assumptions
* Calculates approximate revenue per show

### Footfall Analysis

* Tracks audience trends over time
* Displays footfall using a line graph
* Detects decreasing trends in audience
* Generates warnings for consistent decline

## Use Cases

* Theatre owners to monitor occupancy
* Producers to estimate performance
* Analysts to study audience trends
